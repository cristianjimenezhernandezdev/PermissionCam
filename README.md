# PermissionCam

Projecte Android per gravar vídeo (MP4) o fer fotos amb permisos gestionats i guardat segur.

## Com clonar i executar

1. Clona el repositori:
   ```
   git clone https://github.com/cristianjimenezhernandezdev/PermissionCam.git
   ```
2. Obre la carpeta amb Android Studio.
3. Deixa que es resolguin les dependències (Gradle sync).
4. Compila i executa a un dispositiu o emulador amb càmera.

## Requisits
- Android Studio (recomanat: versió estable més recent)
- Dispositiu/emulador amb càmera
- Permisos de càmera i, si cal, d'emmagatzematge (Android 9 o inferior)

## Notes
- El projecte inclou fitxers de configuració i scripts per a una execució immediata.
- Si tens problemes amb permisos, revisa el manifest i la configuració de l'app.
